parseConnectionID

Parses the current connection ID from a buffer. 

void parseConnectionID(
    char *buffer,
    char *connectionID);
Parameters
buffer

The buffer to search.

connectionID

The connection ID to write.

Discussion

Updates connectionID with an ASCII string representing the connection ID, if the buffer contains one. Checks the first seven bytes for the connection ID. Writes a null byte if unable to parse.

There are four instances in which this can be called: when a CONNECT command is processed, when a CONNECT FAIL command is processed, when a CLOSED command is processed, and when a +IPD command is processed.

rx

Reads a single serial message, up to but not including \n, into serialBuffer. 

size_t rx();
Return Value

The number of bytes read. 0 if no data was read.

Discussion

Guarantees null termination of serialBuffer.
