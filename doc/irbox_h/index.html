BAUD_RATE

Use this baud rate for all serial communication (but not bitbanging). 

#define BAUD_RATE 9600
Discussion

Valid values: any baud rate the board works with.

INIT_DELAY

A short delay at startup, in milliseconds, to wait for connected devices to become initialized. 

#define INIT_DELAY 2000
Discussion

Valid values: 0 ≤ n ≤ ULONG_MAX.

IR_RX_PIN

The pin to use to receive infrared data. 

#define IR_RX_PIN 11
Discussion

Valid values: an Arduino digital pin.

IR_TX_PIN

The pin to use to transmit infrared data. Recommend pin 3 for best results. 

#define IR_TX_PIN 3
Discussion

Valid values: an Arduino digital pin.

SERIAL_BUFFER_SIZE

The size, in bytes, of the buffer to use to read serial data. 

#define SERIAL_BUFFER_SIZE 62
Discussion

Valid values: 1 ≤ n ≤ INT_MAX.

Arduino UART buffer size is 64, so recommend two bytes smaller than that for reasons I don't fully understand. Note that this results in a maximum command length. This yields a maximum effective command length of 51 bytes, which is plenty for even the most complex tx.

SERIAL_TIMEOUT

The timeout, in milliseconds, to pass to Serial.setTimeout(). 

#define SERIAL_TIMEOUT 10
Discussion

Valid values: 0 ≤ n ≤ LONG_MAX.
